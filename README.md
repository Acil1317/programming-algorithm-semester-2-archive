# Programming and Algorithm Semester 2 Archive

Repositori ini dibuat untuk mendokumentasikan seluruh tugas dan latihan praktikum pada mata kuliah Algoritma dan Pemrograman Semester 2. Dokumentasi ini disusun secara bertahap untuk mencatat perkembangan pemahaman saya dalam bahasa pemrograman C++.

## Struktur Materi dan Dokumentasi

Pengerjaan repositori ini dibagi menjadi empat bagian utama sesuai dengan kurikulum pembelajaran:

### Bagian 1: Logika Dasar dan Manipulasi Array
Fokus pada penguatan logika pemrograman dasar serta penggunaan array satu dimensi dan dua dimensi dalam penyelesaian masalah.
File utama: Latihan pertama.cpp, training.cpp, Array 2D.cpp, Array_contoh.cpp.

### Bagian 2: Pemrograman Modular dan String
Mempelajari teknik pemecahan kode menjadi fungsi (Modular Programming), konsep passing variable, serta pengolahan tipe data teks atau string.
File utama: Function.cpp, Passing Variable.cpp, fungsi string.cpp, QT_String.cpp.

### Bagian 3: Struct dan Tipe Data Bentukan
Pendalaman mengenai penggunaan Struct dan Nested Struct untuk mengelola data yang lebih kompleks dan terorganisir.
File utama: QT_Struct.cpp, Tugas Presensi Nested Struct.cpp, QT_Class.cpp, QT_Bonus.cpp.

### Bagian 4: Object Oriented Programming (OOP)
Penerapan prinsip pemrograman berorientasi objek serta penyelesaian tugas-tugas besar sebagai evaluasi akhir semester.
File utama: OOP.cpp, Tugas OOP.cpp, Data Tanaman OOP.cpp, serta file tugas final lainnya.

## Informasi Teknis
Seluruh kode dalam repositori ini ditulis menggunakan bahasa C++ dengan standar compiler modern. Pengelolaan versi file dilakukan menggunakan Git untuk memastikan setiap tahapan pengerjaan tercatat dengan baik.

Dibuat sebagai bagian dari arsip akademik semester 2.
